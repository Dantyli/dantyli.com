# dantyli.com
### 应用技术
前端：`react`+`redux`+`scss` 使用`create-react-app`构建   

服务端：`koa2`+`koa-router`   

### 本地运行
* 安装模块 `npm i`

* 启动服务 `npm run server`

* 启动前端项目 `npm start`

### 项目介绍
定时抓取数字货币信息平台数据，展示前100条信息，主要信息如下：   

| 编号 | 币种 | 市值 | 全球指数(¥) | 24h成交 | 24h涨幅 |
| :------: | :------: | :------: | :------: | :------: | :------: |
